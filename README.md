# test_fastapi_CRUD

Levantar API con FASTAPI que inserte registros en una sqlite a través de POST.
Insutrcciones de ejemplo:
1. Instalar requirements.txt, ```$ pip install -r requirements.txt```
2. Ejecutar main.py para levantar el servicio web: ```$ uvicorn main:app --reload```
3. Hacer un POST con request e insertar un ejemplo en la bbdd, ejecutando test_api.py
4. A través de dbviwer, abrir y visualizar la bbdd de la sqlite.
